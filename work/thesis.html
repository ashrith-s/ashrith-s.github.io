Tangible Web Design for Blind People

A tangible method to design web layouts

Abstract

Blind and visually impaired people can consume web content using magnifiers or screen readers but there is no way for them to design the layout of websites. I aim to conduct semi-structured interviews followed by co-design sessions with blind and visually impaired web design aspirants to understand the best approaches to design a tangible input device using which they can create and edit the structural layout of a webpage, and eventually implement this design and run usability tests.
